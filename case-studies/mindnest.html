MindNest

35% reduction in week-one churn through improved onboarding — redesigning the habit formation experience for a wellness app.

MN

The Problem

52% of users churned within the first 7 days, before experiencing the app's core value proposition. New users felt overwhelmed by options and unclear about how to build a consistent wellness routine.

Constraints

  • Must work offline for core features
  • Content library already established

Discovery & Insights

Analysed cohort data to identify churn patterns. Conducted diary studies with 8 users over 2 weeks. Found that users who set reminders had 3x better retention.

Key Learnings

🔔
Push notification timing
Would have tested notification timing more rigorously for optimal engagement
👫
Social features matter
Could have added more social features earlier to increase accountability
🎉
Celebration moments
Matter more than gamification points — small wins keep users motivated
🎯
Small, achievable goals
Key to habit formation — don't let users set unrealistic expectations
